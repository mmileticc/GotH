About GotH

What is GotH?

GotH (Guitar to Harmonica) is a free online tool for musicians who want to play guitar melodies on the harmonica. It converts guitar tabs into harmonica tablature in real-time — no account, no installation needed.

The project is run by a single developer-musician. It has visitors from all over the world, which is amazing and very motivating — thank you all!

Why GotH?

The logical acronym would be GTOH — but GotH sounds cooler, is easier to remember, and — let’s be honest — programmers love a good dark mode.

If Java can be named after coffee and Python after a British comedy troupe, GotH proudly continues the fine tradition of questionable naming conventions in software engineering.

The Problem It Solves

Many songs have iconic guitar riffs that sound amazing on a harmonica — but guitar tabs (expressed in frets) and harmonica tabs (expressed in blow/draw holes) speak two different languages. GotH bridges that gap instantly.

GotH is currently in beta. Core features are stable, but optimizations, expanded harmonica type support, and UI improvements are still in progress. Every piece of feedback directly shapes development.

Planned Features

  • Support for chromatic and tremolo harmonicas
  • Import tabs from text files
  • Audio preview — note playback
  • Improved mobile interface
